How to use
1.Decompress

2.change the directory to where main.cpp exists

3.type g++ main.cpp in command line and you will see a.out in the folder

4.type ./a.out in_kiss out_kiss out_dot
e.g., ./a.out input.kiss output.kiss output.dot 

And then you will see a file named input.dot (before minimization)

two files named output.kiss and output.dot (after minimization)
